Les bases HTML : les media

Les images

Par défaut, on peut afficher une image à l'aide de la balise <img />.

Avec HTML5, une balise d'encadrement de media a été créé, <figure>. Figure peut servire de cadre, et elle possède en plus une balise enfant optionnelle, <figcaption>, qui permet d'ajouter une légende à l'image.

La balise img

Les proportions

Attention, si on n'indique qu'une largeur à une image, la hauteur se recalcule automatiquement.

Si on veut réduire une taille en imposant une hauteur et une largeur, il faudra respecter les proportions (aussi nommé "l'homothétie"), le rapport hauteur / largeur

L'audio

la balise <audio></audio> fonctionne sur le même principe quela balise vidéo : on peut soit intégrer la source du fichier directement dans la balise ou dans une ou plusieurs balises <source/> filles.

Les vidéos

Les vidéos hébergées sur le site

On utilisa la balise <video></video>

il est possible de définir la source de la vidéo directement dans la balise

Mais les vidéos possèdent aussi une balise fille <source />

Utiliser la balise source permet de proposer plusieurs format de vidéos (mp4, webm, avi), ce qui permet d'adapter le format aux formats qu'accepte le navigateur.
